# Proxier
Proxier is a library to provide free access Proxy server's hosts as code.
Please, check https://proxier.io/.

## What does it do
It consumes data provided by `proxier.io` APIs.

## How to install?
  $ pip install proxier

## How to use it?

```python
import proxier
proxy_ip, proxy_port = proxier.get_proxy()
```
